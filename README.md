# Soil Moisture Prediction Workflow

This repository contains a reproducible workflow for exploring soil moisture prediction using weather, soil, and satellite-derived variables.

## Research question

Can weather, soil properties, and satellite-derived indicators improve prediction of surface soil moisture across agricultural fields?

## Planned workflow

1. Obtain public weather, soil, and satellite-related datasets.
2. Clean and harmonize input data.
3. Develop predictor variables.
4. Train and evaluate machine-learning models.
5. Create maps, tables, and figures.

## Project structure

- `data/raw/`: original input data, not tracked if files are large or restricted
- `data/processed/`: cleaned data used for analysis
- `notebooks/`: exploratory analysis notebooks
- `src/`: reusable Python scripts
- `outputs/figures/`: final figures
- `outputs/tables/`: final tables
- `docs/`: workflow diagrams and supporting documentation
